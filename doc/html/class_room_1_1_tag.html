tag : classe de représentation d'un AprilTag

Author

GOBERT Grégoire
ROUX Marie
Date

Un Tag décrit un AprilTag dans une pièce :
-> Famille et id de famille
-> Pièce qu'il sert à décrire
-> Taille réelle du tag imprimé
-> Position du tag dans la pièce
-> Orientation du tag dans la pièce
