Detailed Description

Human : classe de représentation d'un être humain.

Author
BASSO-BERT Yanis
GOBERT Grégoire
ROUX Marie
Date
2020

Un Human décrit un être humain :
-> Positionnement et orientation de sa tête dans la pièce
-> Direction de son regard
-> Rayon reptésentant le trajet du regard (voir classe Ray)
-> Un eye-tracker qui lui est propre (voir classe EyeTracker)
-> Un ensemble de déclencheurs d'action (voir classe Trigger)
