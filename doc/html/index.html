EyeDoIt - Utilisation

Matériel nécessaire

Un EyeTracker Pupil Labs core
Des AprilTags imprimés sur papier
Un oridnateur assez puissant pour gérer le processus
Du matériel d'interaction (souris, micro, etc)
Des objets connectés
De la connectique (câbles)

Installation

Etape 1 : Disposer les AprilTags dans la pièce

Etape 2 : Configurer l'application dans "Application.py"

  • Instanciation d'un objet Application

  • Ajout d'une ou plusieurs pièces via addRoom
  • Ajout d'un ou plusieurs tags par pièce via addTag
  • Ajout d'un ou plusieurs objets connectés par pièce via addConnectedObject
  • Ajout d'une ou plusieurs spheres par objet via addBoundingSphere
  • Ajout d'une ou plusieurs actions par sphere via addAction

  • Ajout d'une ou plusieurs actions par sphere via addAction

  • Ajout d'un ou plusieurs humain via addHuman
  • Ajout d'un ou plusieurs triggers par humain via addTrigger

Etape 3 : Lancement des différents programmes dans cet ordre

  • Lancer le logiciel "Pupil Capture" et veiller, dans les paramètres, à desactiver le "Frame Publisher"
  • Réaliser une calibration de l'EyeTracker depuis "Pupil Capture" si nécessaire
  • Lancer sur un noyau les programmes occupant la fonction de triggers (entrée de la ToolKit)
  • Lancer sur un autre noyau les programmes faisant la liaison avec les objets connectés (sortie de la ToolKit)
  • Lancer sur un autre noyau "Application.py" avec l'application précédemment configurée.

Etape 4 : Fin d'utilisation : arrêter les différents processus lancés
